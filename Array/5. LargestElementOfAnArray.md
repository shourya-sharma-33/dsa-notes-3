Brute -> Better -> Optimal

Brute : we can sort this and the last element (nlogn)
Optimal : first element is largest and compare , if greater we replace

```cpp
int largest(vector<int> &arr, int n)
{
    int largest = arr[0];
    for(int i = 0; i<n; i++)
    {
        if (arr[i]>largest)
        {
            largest = arr[i];
        }
    }
    return largest;
}
```
